주요 기능

프로그램 정보 다이얼로그

.NET 8 Runtime 설치 안내
K-Zone 실행에는 .NET 8 Runtime (x64)이 필요합니다.
설치되어 있지 않으면 실행 시 자동으로 설치 안내가 뜹니다.

설치 시 Windows 보안 경고
EXE 실행 시 SmartScreen 경고가 뜰 수 있습니다.
"추가 정보" → "그래도 실행" 클릭하면 실행됩니다.
